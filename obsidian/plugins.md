# Obsidian Plugin Setup

## Enable Community Plugins

1. Open Obsidian
2. Settings > Community plugins > Turn on community plugins

## Required Plugins

### Dataview
- Search "Dataview" in Community plugins
- Install and enable
- Powers all dashboard queries in this vault

### Templater
- Search "Templater" in Community plugins
- Install and enable
- Set template folder to `templates/` in Templater settings

## Recommended Plugins

### Shell Commands
- Enables running scripts from within Obsidian
- Useful for triggering status updates manually

### Obsidian Skills (by kepano)
- Install via: copy from https://github.com/kepano/obsidian-skills into `.claude/` in your vault
- Teaches Claude agents to write proper Obsidian-flavored markdown
